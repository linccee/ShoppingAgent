# 镜澜导购 (Sales Agent) - 项目介绍

## 1. 项目概述

**镜澜导购**是一款多平台AI购物助手，帮助用户在 Google Shopping、Amazon 和 eBay 三大平台比较商品价格、汇总_reviews、分析情感、换算货币，并生成购买建议。该应用具有跨会话的持久化对话记忆功能。

### 核心功能

| 功能 | 描述 |
|------|------|
| 多平台搜索 | 同时搜索 Google Shopping、Amazon、eBay 商品 |
| 价格比较 | 实时获取各平台最新价格 |
|  Reviews分析 | 提取_reviews要点，分析情感倾向 |
| 货币换算 | 支持 USD、EUR、GBP、JPY、CNY、KRW 等主流货币 |
| AI推荐 | 基于价格、_reviews、平台信誉生成购买建议 |
| 会话记忆 | 支持多轮对话，上下文持久化到MongoDB |

## 6. LangGraph Agent 模块详解

### 6.1 State 定义

`SharedState` (TypedDict) 是贯穿整个StateGraph的中央状态对象:

| 字段 | 类型 | 说明 |
|------|------|------|
| `user_query` | str | 用户输入 |
| `session_id` | str | 会话ID |
| `messages` | list[BaseMessage] | 消息历史 |
| `search_results` | SearchResult | 搜索结果 |
| `candidate_products` | list[dict] | 候选商品列表 |
| `price_info` | list[PriceInfo] | 价格信息 |
| `review_analyses` | list[ReviewAnalysis] | Reviews分析结果 |
| `currency_result` | dict | 货币转换结果 |
| `recommendations` | list[Recommendation] | 推荐结果 |
| `final_response` | str | 最终响应 |
| `workflow_status` | str | 工作流状态 |
| `steps` | list[WorkflowStep] | 执行步骤追踪 |

### 6.2 工作流图

**默认并行图**:
```
input → search → ┬→ price ─┐
                 ├→ review ─┼→ recommend → output
                 └→ currency─┘
```

**简单串行图**:
```
input → search → price → recommend → output
```

### 6.3 节点函数

| 节点 | 函数 | 并行能力 |
|------|------|----------|
| `input_node` | 初始化状态，存储用户查询到messages | - |
| `search_node` | 调用 `search_products` 工具 | 串行 (必须等待) |
| `price_node` | 获取候选商品实时价格 | 可并行 |
| `review_node` | 分析商品_reviews | 可并行 |
| `currency_node` | 检测货币并转换预算为USD | 可并行 |
| `recommend_node` | 商品排序并生成推荐报告 | - |
| `output_node` | 格式化最终响应 | - |

### 6.4 Agent创建

`agent_core.py` 负责:
- 使用 `MongoDBSaver` 创建LangGraph ReAct Agent
- `stream_agent()` 在后台线程运行，通过队列yield token/工具事件
- 支持流式输出

`factory.py` 工厂模式:
- `create_shopping_agent()` 返回 ReAct Agent 或 StateGraph Agent

## 7. 工具模块详解

### 7.1 搜索工具 (`tools/search_tool.py`)

**功能**: 多平台商品搜索

| 平台 | 引擎 | SKU来源 |
|------|------|---------|
| Google Shopping | `google` (tbm=shop) | - |
| Amazon | `amazon_product` | ASIN |
| eBay | `ebay_product` | product_id |

**搜索规则**:
- 使用英文关键词搜索
- 不添加日期/年份限制

### 7.2 价格工具 (`tools/price_tool.py`)

**功能**: 获取商品实时价格

| 平台 | 数据来源 |
|------|----------|
| Amazon | `product_results.price` |
| eBay | `buy.buy_it_now.price` |

### 7.3 Reviews工具 (`tools/review_tool.py`)

**功能**: 情感分析和Reviews提取

**情感判断**:
- 评分 ≥ 4.0 → 正面
- 评分 ≤ 2.5 → 负面
- 其他 → 中性

**数据提取**:
- Amazon: rating, reviews数量, summary, 评论列表
- eBay: reviews数量, 评论列表 (最多10条)

### 7.4 货币转换工具 (`tools/currency_exchange_tool.py`)

**API**: exchangerate-api.com (v6)

**支持的货币**: USD, EUR, GBP, JPY, CNY, AUD, CAD, INR, KRW

**货币检测** (从用户查询):
- `¥` / `人民币` / `元` / `CNY` → CNY
- `£` / `英镑` → GBP
- `€` / `欧元` → EUR
- `$` / `美元` → USD

**预算提取**: 支持 `¥3000`, `$500`, `预算: 3000` 等格式

**默认假设**: 中文 → CNY，英文 → USD

### 7.5 Tavily工具 (`tools/tavily_tool.py`)

**功能**: 实时网络搜索和URL内容提取

| 工具 | 用途 |
|------|------|
| `tavily_search` | 获取最新/时效性信息 |
| `tavily_extract` | 从URL提取内容 |

### 7.6 工具并行规则

| 工具 | 规则 |
|------|------|
| `search_products` | 必须串行，每步一次调用 |
| `prices` | 可并行，同工具最多3个并发 |
| `analyze_reviews` | 可并行 |
| `currency_exchange` | 可并行 |

## 8. 数据库设计

### 8.1 MongoDB 集合

| 集合名 | 用途 |
|--------|------|
| `users` | 用户账户信息 |
| `sessions` | 聊天会话 (包含消息、token使用、用户ID) |
| `compressed_agent_states` | 压缩后的LLM状态 (用于记忆优化) |
| `compression_failed_tasks` | 压缩失败任务 (用于重试) |
| `compression_degradation_state` | 降级追踪 (按线程) |

### 8.2 检查点机制

- 使用 `langgraph.checkpoint.mongodb` 的 `MongoDBSaver`
- 被 `CompressedCheckpointer` 封装以支持状态压缩
- 会话隔离通过 `thread_id` = `session_id` 实现

## 9. 认证流程

### 9.1 注册流程

```
用户提交注册 → 密码强度验证 → bcrypt加密 → 存入MongoDB → 返回成功响应
```

### 9.2 登录流程

```
用户提交登录 → 验证密码 → 生成JWT令牌 → 返回access_token和用户信息
```

### 9.3 JWT令牌结构

| 声明 | 内容 |
|------|------|
| `sub` | user_id |
| `username` | 用户名 |
| `iat` | 签发时间 |
| `exp` | 过期时间 |

### 9.4 前端认证状态

- Token存储在 `localStorage` (`access_token`, `user`)
- `AuthContext` 管理认证状态
- 401响应自动清除存储并触发登出

## 15. 项目特色

### 15.1 AI驱动的工作流

使用 LangGraph 实现结构化购物助理工作流，支持并行工具调用和状态持久化。

### 15.2 多平台聚合

通过 SerpAPI 统一封装 Google Shopping、Amazon、eBay 三大平台的搜索、定价、_reviews API。

### 15.3 智能货币转换

自动检测用户输入中的货币类型，支持主要货币间的实时转换。

### 15.4 记忆压缩
